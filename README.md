<h1>Инструкция для запуска приложения</h1>

<h2>Создайте и активируйте окружение</h2>

<p>conda create --name env_pr python=3.6</p>

<p>conda activate env_pr</p>

<h2>Установите необходимые зависимости</h2>

<p>pip install -r requirements.txt</p>

<h2>Запустите приложение</h2>

<p>python main.py</p>
